Name : Greeny Organic

This project aims to develop an interactive and user-friendly interface for Greeny Organics for easy browsing, ordering, and communication with customers interested in Nigerian cuisine. The purpose of this project is to improve customer interaction and make the ordering process for Greeny Organics efficient through UI/UX Design.

Goals:

  • Provide high-quality Nigerian food items (frozen, fresh, dried) to customers aged 18-65.
  • Make Nigerian cuisine more accessible by enabling ordering and delivery from home.
  • Enhance customer interaction and streamline the ordering process through a user-friendly mobile app.

Outcomes:

  • Create a nostalgic and comfortable user experience through the app's UI/UX design.
  • Facilitate direct ordering for customers to browse and purchase items seamlessly.
  • Improve communication channels like email and WhatsApp to better support customer engagement and inquiries.
  • Leverage technology to offer an immersive experience and make authentic Nigerian cuisine easily accessible.

Deliverables:

  • Mobile application interface
  • Content for the platform
  • Communication tools
